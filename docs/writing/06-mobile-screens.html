La Tarea

Mi familia y yo

A1 · 180–220 palabras · Unidad 5
Misión
Escribe una composición sobre tu familia y lo que les gusta hacer. Describe a tres miembros usando posesivos y el verbo gustar.
Requisitos
  • ▸ 3 posesivos (mi, tu, su…).
  • ▸ Conjuga gustar.
  • ▸ 5 palabras de familia.
  • ▸ Conectores: y, pero, porque.
"La familia" es singular. Escribe la familia es, no la familia son.
Desliza para escribir

Mi familia son es (concordancia) pequeña pero muy especial. Vivimos en Estepona.

Mi madre se llama Elena y es profesora. A ella le gusta mucho [→ le encanta] leer.

Mi padre es muy simpática [→ muy simpático] y le gusta gustan (sujeto plural) los deportes.

Patrón: concordancia de género y número. Antes de v2, repasa cada sustantivo.
HANDOFF NOTES
Segmented control behaviour
Tarea / Escribir control stays pinned to the top even with keyboard open. Use sticky top-0 + position-relative parent. Swipe gesture uses onTouchStart + onTouchEnd delta > 50px.
State persistence
Switching views must NOT reset scroll position or cursor. Store cursorOffset and scrollTop in component state; restore on tab change.
Autosave pattern
Debounce 3s after last keystroke · fire on blur · fire on route change via useEffect cleanup · fire on visibilitychange hidden · queue to IndexedDB when offline (reuse Piñata's existing offline patterns).
Correction render
Don't use dangerouslySetInnerHTML. Iterate segments[] and render typed React elements per kind. Keep each note element in-flow (parenthetical), never absolute-positioned.
Accessibility
Radio cards: role="radiogroup", arrow keys cycle, space/enter selects. Sheet: aria-modal="true", focus trap, return focus to trigger on close. Colour alone never carries correction severity — each segment also has a typed badge/note.
